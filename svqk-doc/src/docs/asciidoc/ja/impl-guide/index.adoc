= 実装ガイド

本書は、本プロジェクトにおける実装作業を効率的かつ一貫性を持って進めるための指針を提供することを目的としています。

本書は、本プロジェクトに参画する開発者を対象としています。開発者が本書に示されるプロセス、及びアーキテクチャ記述に示される標準を遵守することにより、高品質なシステムを効率的に開発することを目指します。

== ガイドの読み進め方

本書は以下の順序で読み進めることを推奨します。
この順序は、開発者が実際に動くアプリケーションとソースコードを手元で参照しながら、本書、及びアーキテクチャ記述の要点を確認できるよう構成されています。

. *環境構築* +
  <<#setup-process>>に従い、開発に使用するPC上に実行・開発環境を構築します。
. *アプリケーションの確認* +
  構築した環境でアプリケーションを実行します。
  アーキテクチャ記述のxref:../arch-desc/index.adoc#usecase[ユースケース]と照らし合わせてアプリケーションを操作し、実装されている機能を確認します。
. *アプリケーションアーキテクチャの確認* +
.. *論理的な構成* +
    アーキテクチャ記述のxref:../arch-desc/index.adoc#application-structure[アプリケーション構成]、xref:../arch-desc/index.adoc#component-structure[コンポーネント構成]を参照し、アプリケーションを構成するコンポーネントとその責務を確認します。
.. *物理的な実装* +
    <<#dev-process>>とそこで説明されている各コンポーネントのソースコードを参照し、コンポーネントの実装方法を確認します。
